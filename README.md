# MobaXterm_Keygen
MobaXterm_Keygen

自动检测MobaXterm安装路径，可以手动选择安装路径，增加了许可证生成功能，可以自动将生成的许可证生成到MobaXterm安装路径

Automatically detects the MobaXterm installation path, allows manual selection of the installation path, adds a license generation feature, and can automatically generate the license into the MobaXterm installation path.
